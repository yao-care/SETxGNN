# SETxGNN - Sverige: Repositionering av Lakemedel

[![Website](https://img.shields.io/badge/Website-setxgnn.yao.care-blue)](https://setxgnn.yao.care)
[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)

Prediktioner for repositionering av lakemedel (drug repurposing) for Sverige med hjalp av TxGNN-modellen.

## Ansvarsfriskrivning

- Resultaten fran detta projekt ar avsedda enbart for forskningsandamal och utgor inte medicinsk radgivning.
- Kandidater for repositionering av lakemedel kraver klinisk validering fore tillampning.

## Projektoversikt

| Element | Antal |
|---------|-------|
| **Lakemedelsrapporter** | 108 |
| **Totala Prediktioner** | 3,672,147 |

## Prediktionsmetoder

### Kunskapsgrafmetod (Knowledge Graph)
Direkt forfragan av lakemedel-sjukdomsrelationer i TxGNN-kunskapsgrafen, identifiering av potentiella repositioneringskandidater baserat pa befintliga kopplingar i det biomedicinska natverket.

### Djupinlarningsmetod (Deep Learning)
Anvander den fortranade TxGNN neurala natverksmodellen for att berakna prediktionspoangar, och utvarderar sannolikheten for nya terapeutiska indikationer for godkanda lakemedel.
